# .agents Directory

This directory contains agent-operational materials for continuing implementation without prior conversation context.

## Contents
- `findings/TEMPLATE.md`: evidence template for final PoC findings.
- `skills/`: repo-local skills required by this project (`proto`, `moon`).
- `skills/README.md`: instructions to refresh repo-local skills.

## First read order for a new agent
1. `AGENTS.md`
2. `TASKLIST.md`
3. `PRD.md`
4. `README.md`
5. `.agents/findings/TEMPLATE.md`

After reading, resume from the `NEXT_TASK_ID` in `TASKLIST.md`.
